These recordings and screenshots were done on OpenSpace 0.21.3+ daily builds 
and some on single precision builds on Mac (those which did not need atmosphere effects).

Since OpenSpace on Linux with NVidia GTX 1050 would crash after 800 or 1400 frames of 4K fisheye screenshots, 
asked Claude.ai to split some recordings into 10 second chunks, which we could use with
"save frames" without crashes at 60 fps.

Later, found that disabling CEFgui and closing all other apps etc would allow 
screenshots to be saved without crashes on OpenSpace on Linux with NVidia GTX 1050.
